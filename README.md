## 2. بوت تليجرام لتصحيح الأخطاء الإملائية (Arabic Spelling Corrector Bot)
نظام متكامل مبني باستخدام منصة n8n ومدمج مع نموذج الذكاء الاصطناعي Google Gemini لتلقي النصوص العربية من تليجرام، وتصحيحها إملائياً ونحوياً بشكل ذكي وسريع.

### الملف الخاص به:
* Malakcorrection.json

### ميزات النظام:
* Telegram Trigger: يستقبل الرسائل والنصوص مباشرة من المستخدمين عبر تليجرام.
* AI Agent Node: معالجة ذكية وفهم سياق الجملة لتصحيح الأخطاء الشائعة (مثل الهمزات والتاء المربوطة).
* Automated Reply: إعادة إرسال النص المصحح تلقائياً للمستخدم فوراً.
